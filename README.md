# taptap-server

## 前言

该项目为taptap页面逻辑后台，基于golang开源框架（zinx-socket版本） [传送门](https://github.com/aceld/zinx)

## 项目介绍

`taptap-server`是一个仿taptap的页面逻辑后端项目，基于golang实现，包括不限于通信、权限、登录...

### 项目布局

``` lua
app-server -- 源码目录
├── common -- 通用函数
├── configs -- 配置文件
├── app     --  后端逻辑代码
├── iserverface -- 接口层
├── server -- 实现层
├── vendor -- 依赖包
├── zlog --   日志

```
